How it WORKS!?

it starts with information gaved by the user the website forms a link like so:

https://web.whatsapp.com/send/?phone=PHONENUMBER&text=MESSAGE&type=phone_number&app_absent=0

PHONENUMBER: this will be replaced with the phone number provided by the user.

MESSAGE: this will be replaced with the message provided by the user.

type=phone_number: This parameter specifies the destination type which is always a phone number.

app_absent=0: This parameter indicates whether the WhatsApp application is installed. A value of 0 indicates that the application may be present, instructing the browser to attempt to open it if possible.
